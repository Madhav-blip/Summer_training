We Learned about git and github. 
Steps to be followed:
1. Install Git as well as VS code.
2. Create a local repository.
3. Add the codes in local repo with code editor and commit them.
4. Create a remote repository on github.
5. Push the code in the remote repo. 
6. You can view your code on github.
